# Submission Checklist

1) ZIP of source code (no .venv, no instance DB)
2) Evaluation report (<= 1000 words)
3) Video demo (<= 15 minutes)
4) Evidence included:
   - evidence/pytest_results.txt
   - evidence/pytest_results_verbose.txt
   - evidence/junit.xml (optional)
   - git log screenshot or git history visible on GitHub
